# README

This README tells you how to use this app in Windows with Python3.

1 select cmd terminal

2 cd FlaskMarket

3 python -m venv venv (only first time)

4 .\venv\Scripts\activate

5 pip install -r ./config/requirements.txt (only installing something)

6 pip freeze > ./config/requirements.txt (only installed something)

7 python run.py

8 pytest test.py (when testing application)

9 CTRL + C

10 deactivate
